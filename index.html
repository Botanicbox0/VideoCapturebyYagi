# 🎬 비디오 프레임 캡처 도구 - 배포 가이드 ## 📥 1단계: 파일 다운로드 아래 전체 코드를 복사하세요: ```html 비디오 프레임 캡처 도구
``` ## 📝 2단계: 파일 저장하기 1. **Windows 사용자**: - 메모장(Notepad) 열기 - 위 코드 전체 복사해서 붙여넣기 - "파일" → "다른 이름으로 저장" - 파일 이름: `index.html` - 파일 형식: "모든 파일 (*.*)" - 인코딩: UTF-8 - 저장 위치: 바탕화면이나 원하는 폴더 2. **Mac 사용자**: - TextEdit 열기 - "포맷" → "일반 텍스트로 만들기" - 위 코드 붙여넣기 - 저장: `index.html` ## 🧪 3단계: 로컬에서 테스트 1. 저장한 `index.html` 파일을 **더블클릭** 2. 브라우저에서 열림 3. 비디오 파일 업로드해서 **제대로 작동하는지 테스트** 작동하면 다음 단계로! ## 🚀 4단계: GitHub Pages에 배포 ### A. GitHub 저장소 만들기 1. https://github.com 로그인 2. 오른쪽 위 `+` → "New repository" 3. Repository name: `video-frame-capture` 4. Public 선택 5. "Add a README file" 체크 6. "Create repository" 클릭 ### B. 파일 업로드 1. 저장소 페이지에서 "uploading an existing file" 링크 클릭 2. `index.html` 파일을 드래그 앤 드롭 3. "Commit changes" 클릭 ### C. GitHub Pages 활성화 1. 저장소 상단 "Settings" 클릭 2. 왼쪽 메뉴 "Pages" 클릭 3. Source: "Deploy from a branch" 선택 4. Branch: `main`, `/ (root)` 선택 5. "Save" 클릭 ### D. 완료! 🎉 - 1-2분 후 사이트가 배포됩니다 - 주소: `https://[당신의아이디].github.io/video-frame-capture` - 페이지 새로고침하면 초록색 박스에 주소 표시 --- ## ✅ 체크리스트 - [ ] 코드 복사 - [ ] `index.html`로 저장 - [ ] 더블클릭해서 브라우저에서 테스트 - [ ] 비디오 업로드 및 캡처 테스트 - [ ] GitHub 저장소 생성 - [ ] 파일 업로드 - [ ] GitHub Pages 활성화 - [ ] 배포된 사이트 확인 막히는 부분이 있으면 스크린샷과 함께 알려주세요!
